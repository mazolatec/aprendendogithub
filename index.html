Remote Commands
      git remote add origin URL 
      Add a remote repository
      # Conecta um repositório remoto
    
      git push 
      Push changes to a remote repository
      # Envia commits para o repositório remoto
    
      git pull 
      Fetch and integrate changes from a remote repository
      # Atualiza o repositório local com mudanças remotas
    
      git clone URL 
      Clone a repository
      # Clona um repositório para o diretório local
Advanced Commands
      git log 
      Show commit logs
      # Mostra o histórico de commits
    
      git diff 
      Show changes between commits or working tree
      # Compara mudanças entre commits ou o estado atual
    
      git stash 
      Stash changes in a dirty working directory
      # Salva temporariamente mudanças pendentes
    
      git reset 
      Reset the current HEAD to a specific state
      # Reverte para um estado anterior
